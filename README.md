# Insurance Pricing Hackathon Project
**3rd place overall (out of 25 teams) and 1st place in prediction task**

This repository contains a team machine learning project developed during a data science hackathon, focused on predicting insurance prices and analyzing competitor pricing strategies. 

## Project Goals
- Predict competitor insurance prices using customer and vehicle data  
- Analyze pricing behavior across different insurers  
- Identify market opportunities and pricing gaps  

##  Repository Structure
- `notebooks/` – Jupyter notebook containing data analysis, feature engineering, and modeling  
- `src/` – Python scripts (main.py)  
- `presentation/` – Final team presentation
